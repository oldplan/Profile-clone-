User Journey Flows

Step-by-step navigation paths for each role

👤 Normal User Journey

01
/start → Mode selection screen appears
02
Tap 👤 Normal User → Main menu loads
03
👤 Accounts → Add & login Telegram account via phone
04
🔄 Fetch Groups → Bot fetches all joined groups
05
🎯 Select Groups → Toggle target groups, save
06
✏️ Set Message → Input broadcast message
07
⏳ Set Delay → Set seconds between messages
08
▶️ Start Loop → Broadcast begins

⚙️ Advanced User Journey

01
/start → Mode select → pick ⚙️ Advanced
02
👤 Accounts → Add multiple accounts
03
📌 Specific Groups → Add groups by ID for precision targeting
04
📡 Log Forwarding → Setup live log channel
05
🗓 Scheduler → Set interval / daily / weekly schedule
06
⏱ Set Interval → Fine-tune burst timing
07
▶️ Start Loop → Choose: Specific / Selected / Both
08
Monitor via log forwarding in real-time

🛡 Admin / Owner Journey

01
Bot auto-detects OWNER_ID match on /start
02
Extra buttons appear: 🛡 Grant + 📊 Stats
03
Grant Permission → Enter user ID, choose plan level
04
Written to permissions.json instantly
05
📊 Permission Stats → View all users + their access level
06
Share 💎 Plans keyboard → links to owner Telegram
07
🔥 Reset All → Nuclear option with confirmation gate
08
Manage config.py directly for bot settings
